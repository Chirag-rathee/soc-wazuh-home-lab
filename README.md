# soc-wazuh-home-lab

#### \#Overview

This project demonstrates a SOC home lab built using Wazuh SIEM for monitoring,

log analysis, attack detection, and security event investigation.

The lab simulates real-world cyberattacks such as SSH brute force and Nmap

reconnaissance scanning to generate alerts and analyze malicious activity.

\---

#### \## Objectives

\- Build a beginner-friendly SOC home lab

\- Detect SSH brute force attacks

\- Detect Nmap reconnaissance scans

\- Create custom Wazuh detection rules

\- Monitor alerts in real time

\- Practice security monitoring and incident investigation



#### \## Tools Used

| Tool | Purpose |

|---|---|

| Wazuh | SIEM \& Log Monitoring |

| Kali Linux | Attack Simulation |

| Ubuntu Server | Target System |

| Suricata | IDS Monitoring |

| Nmap | Network Scanning |

| Hydra | Brute Force Simulation |

#### \## Lab Architecture
